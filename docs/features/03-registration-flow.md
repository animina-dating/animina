# 3. Registration & Profile Completion Flow

**Status:** Not Started

---

## Flow Steps

1. **Registration** - Fill out the registration form (required + optional fields, including an optional referral code)
2. **Email Validation** - Confirm email via 6-digit PIN (referral credit is processed at this step)
3. **Waitlist** - User is placed on the waitlist; admin grants access or user is auto-activated after 5 successful referrals
4. **Profile Completion** (post-registration):
   - Set up flags (white/green/red)
